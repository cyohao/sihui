# sihui（思绘）

本地优先的轻量白板应用，基于 Excalidraw 封装。打开即用，数据始终保存在本地——浏览器版存于 IndexedDB，桌面版存于本地文件，无需联网、无需账号。

## 开发

```bash
npm install
npm run dev
```

## 构建（Web / PWA）

```bash
npm run build
```

将 `dist/` 部署到任意静态文件服务即可。首次在线访问后，PWA 会缓存应用资源，后续可离线打开，也可安装为本地应用。

## Windows 桌面应用

桌面版基于 Tauri 2，仅加载本地构建资源，不依赖 CDN。

```bash
npm run tauri dev      # 开发调试
npm run tauri build    # 打包安装程序
```

打包产物位于 `src-tauri/target/release/bundle/nsis/`，生成 `sihui_<版本>_x64-setup.exe` 安装程序。

### 数据存储

桌面版会把全部白板自动保存到用户文档目录：

```text
文档\思绘数据\boards.json
```

写入采用「临时文件 → 备份 → 重命名」的原子流程，异常中断可从 `.bak` 自动恢复。工具栏的「数据目录」可直接打开该文件夹；「备份全部」「恢复全部」用于迁移到其他电脑。浏览器版本则使用 IndexedDB。

### 后台驻留与系统托盘

桌面版常驻系统托盘，关闭窗口不会退出程序：

- **点击窗口关闭按钮**：隐藏到托盘，程序继续在后台运行
- **最小化窗口**：从任务栏隐藏，仅保留托盘图标
- **左键单击托盘图标**：唤回主窗口
- **托盘右键菜单**：
  - 显示窗口
  - 退出（真正关闭程序）

后台运行期间自动保存仍然有效，数据不会丢失。

## 思维导图

工具栏「思维导图」可由一段缩进大纲一键生成导图，生成的节点就是普通画布元素，能与手绘内容自由混排，导出 / 保存全部复用。

- **生成**：每行一个节点，用缩进（空格或 Tab）表示层级，一键插入当前白板
- **Tab**：为选中节点添加子节点；**Enter**：添加同级节点——新增后整张图自动重新排版
- **Delete / Backspace**：删除选中节点及其整棵子树
- **双击节点**：修改文字
- **折叠 / 展开**：收起或还原子树（折叠节点以虚线边框标记）
- **整理**：按 tidy-tree 一键重新排版，以根节点为锚点不跳位
- 拖动节点时连线自动跟随；拖动带子节点的节点会带动整棵子树一起移动

数据结构存于元素的 `customData` 中，画布即唯一数据源，无需额外存储。

## 功能一览

- 无限画布，画笔、图形、文本、箭头、图片，撤销 / 重做
- 思维导图：大纲生成、Tab/Enter 增删、自动排版、折叠展开、连线跟随
- 多白板管理：新建、切换、重命名、删除
- 自动保存（防抖落盘，退出前兜底保存）
  - 浏览器：IndexedDB
  - 桌面：本地 `boards.json` 文件
- 导入 / 导出 `.sihui.json`，并兼容已有 `.excalidraw` 文件
- 导出 PNG 和 SVG
- 备份 / 恢复全部白板，便于跨设备迁移
- PWA 安装与离线缓存（Web 版）
- 桌面版托盘驻留，关闭 / 最小化隐藏到后台

## 技术栈

React 18 · TypeScript · Vite 6 · Excalidraw · idb（IndexedDB）· Tauri 2（Rust）· vite-plugin-pwa
